## 태그

```
<태그 속성= "값"></태그>
```

1. "<"와 ">" 사이에 명령어(tag, element) 작성
2. 시작태그와 종료태그로 구성
3. 종료태그가 없는 태그들(br, img, hr, meta, input)
4. 태그는 중첩이 가능하며 종료태그는 나중에 작성한 태그 먼저 작성
5. 가급적 모든 태그와 속성은 소문자로 작성

## 속성의 특징

1. 태그와 속성, 속성과 속성은 공백으로 구분

```html
<a href="#"></a>
```

2. 여러 개의 속성을 사용할 수 있다.

```
<img src "/" width="100">
```
